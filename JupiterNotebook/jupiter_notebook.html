JupiterNotebook

  1. Install and open Visual_Studio_Code
  2. Goto extension and install Python
  3. Press CTRL+SHIFT+P and select jupiter: Create New Blank Jupyter Notebook: to create a new JupiterNotebook
  4. Press CTRL+SHIFT+P and select jupiter: Open in Notebook Editor : to create a new JupiterNotebook
  5. a: to new shell above the current shell
  6. b: to new shell below the current shell
  7. y: to code
  8. m: to markdown
  9. ESC: to command mode
