Make a KaliLinux Persistence USB

Requirements:-

  • KaliLinux.iso
  • PowerISO.exe
  • USB drive (more than 8GB)
  1. Insert your USB drive into PC.
  2. Open PowerISO and goto: Tools\Create Bootable USB Drive (if there is any message, click on OK\Yes.)
    3. Source Image File: Click on three dots and select your ISO image path.
    Destination USB Drive: Select your USB drive.
    Click on Start\OK.
    (Note: a process will be start, wait till finish it.)
  3. After finishing that, close the all windows and PowerISO.
  4. Now boot your PC with USB drive and select Live System.


  5. Press CTRL+ALT+T and type:
    6. sudo fdisk -l
  6. Select and copy your disk in the lists. (/dev/sdb)
  7. Type: sudo fdisk /dev/sdb
  8. Type: n and press ENTER five times
  9. Type: w
  10. Type: sudo fdisk -l (To check your created disk and copy that.)
  11. Type: sudo mkfs.ext4 -L persistence /dev/sdb3
    12. (Remember: /dev/sdb3 is your USB disk.)
    (Note: Wait for a while and let the process completed)
  12. Type: sudo e2label /dev/sdb3 persistence
  13. Type: sudo mkdir -p /mnt/USB
  14. Type: sudo mount /dev/sdb3 /mnt/USB
  15. Type: sudo echo "/ union" > /mnt/USB/persistence.conf
  16. Type: sudo umount /dev/sdb3
  17. Type: sudo init 0 (To shutdown the PC)
  18. Boot your PC with USB drive and select Live system (Persistence, check Kali.org/Prst
  19. Make some changes in your PC settings and Reboot with USB drive and select Live system (Persistence, check Kali.org/Prst
    If all changes are still there then your persistence USB has been created successfully.

Do this after installation of KaliLinux

  1. Connect your PC with Internet
  2. Press: CTRL + ALT + T (to open the Terminal Window)
  3. Type: apt-get update (wait till process completed)
  4. Type: apt-get upgrade (wait till process completed)
  5. Type: apt-get dist-upgrade (wait till process completed)

  6. Install: VisualStudioCode
  7. Install: VisualStudioCode/CodeRunner
