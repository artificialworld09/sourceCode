Visual Studio Code 1.43.0 (text editor)

  1. select word and goto "Change All Occurrences" or "CTRL + F2": to change all same words then press ESC.
  2. CTRL + G "line number": to goto any line by line no.
  3. CTRL + SHIFT + T: to reopen file.
  4. ALT + Down auro: to move Line down.
  5. ALT + Up auro: to move Line up.
  6. SHIFT + ALT + Down auro: to copy line down.
  7. SHIFT + ALT + Up auro: to copy line up.
  8. CTRL + /: to comment/uncomment.

Emmet

  1. p: to paragraph.
  2. b: to bold tag.
  3. i: to italic.
  4. br: to br tag.
  5. div: to make div.
  6. h1: to heading 1 tag.
  7. h.classname: to make a class in h1 tag.
  8. h1.classone.classtwo: to make two classes in h1 tag.
  9. h1#idname: to make a id in h1 tag.
  10. .classname: to make a class in div.
  11. h1#myId.myClass: to make one id and one class in h1 tag.
  12. h1#myId.myClassOne.myClassTwo: to make one class and two id in h1 tag.
  13. ftr: to make footer tag.
  14. hdr: to make header tag.
  15. p{This is paragraph}: to type "This is paragraph" in p tag.
  16. h1*4: to type h1 four times.
  17. p.myClass{This is paragraph}: to type "This is paragraph" in p tag with class.
  18. h1>p: to make child paragraph in h1 tag.
  19. h1>ul>li: to make grand-child list in h1 tag.
  20. h1>ul>li{This is list one}: to type "This is list one" in grand-child list in h1 tag.
  21. h1>ul>li{This is list}*4: to make 4 grand-child list and type "This is list" in h1 tag.
  22. h1>ul>li{This is list $}*4: to make 4 grand-child list and type "This is list" dynamicaly in h1 tag.
  23. h1{This is heading one}>h2{This is heading two}>p{This is paragraph.}: to type sentences with individual child tag.
  24. h1{This is heading one}+h2{This is heading two}+p{This is paragraph.}: to type sentences with individual siblings tag.
  25. h1.myClass+h2.myClass2+p: to type classes with individual siblings.
  26. form: to make a form.
  27. form:post: to make a form with post method.
  28. inp: to input tag.
  29. input:password: to make input with password type.
  30. lorem: to type dummy text.
  31. lorem7: to type dummy text in seven word.
  32. h1>p.myClass>lorem: to type dummy text in child paragraph with a class.
  33. div>ul>lorem5.item*3: to type three times dummy text in list limit with 5 words.
